# Smart Elevator Control System

## 📌 Overview
This project is a smart elevator system built using Raspberry Pi and Python. It combines multiple AI-based authentication methods including face recognition, hand gesture detection, keypad security, and speech recognition to control and secure elevator access.

## 🔥 Project Highlights
- Multi-layer AI authentication system
- Real-time computer vision processing
- Embedded hardware control via Raspberry Pi
- Secure keypad + servo locking mechanism
- Expandable IoT-ready architecture
---

## ⚙️ Features

### 🖐 Hand Gesture Recognition
- Counts fingers using MediaPipe
- Real-time camera detection

### 🎭 Face Recognition
- Identifies authorized users
- Uses face encodings for matching

### 🔢 Keypad Security
- 4x4 keypad input system
- Password validation with servo lock

### ⚙️ Motor Control
- Controls elevator movement up/down
- GPIO-based DC motor control

### 🎤 Speech Recognition
- Captures voice input (basic implementation)

## 🚀 System Idea
The system authenticates the user using multiple methods. If authentication is successful, the elevator system is activated and controlled through motor logic based on user input.
